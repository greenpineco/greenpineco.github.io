What is covered under bug fixes and maintenance?

Bug fixes are changes made to existing code to correct an unintended behaviour.

Maintenance can cover tasks like adding documentation, code cleanup, fixing API integrations, upgrade or replace a 3rd party library, scheduled upgrades to frontend/backend application frameworks, CSS style fixes, copy updates, updating ssl certificates, fixing cross browser issues, testing backups, and more.

If you are unsure if something is covered under these categories please submit a request anyway and we will promptly let you know.
